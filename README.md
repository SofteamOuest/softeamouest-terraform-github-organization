# SofteamOuest Terraform Github Organization

For managing SofteamOuest Github Organization

## Become a member

Edit `modules/people/main.tf`

## Manage projects

Edit `module/projects/main.tf`

## Manage repositories

Edit `module/people/main.tf`

## Manage teams

Edit `module/teams/main.tf`

## Usage 

    terraform init
    terraform plan
    terraform apply
